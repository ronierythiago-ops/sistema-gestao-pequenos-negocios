# Sistema Web de Gestão de Pequenos Negócios

## Descrição
Protótipo Front-end de um sistema para controle de estoque, vendas, clientes e acompanhamento financeiro básico.

## Tecnologias Utilizadas
- HTML
- CSS
- JavaScript

## Funcionalidades Implementadas
- Login
- Cadastro de usuário
- Controle de produtos
- Cadastro de novos produtos
- Visualização de detalhes
- Busca e filtros

## Como Executar
Basta abrir o arquivo `index.html` no navegador.
